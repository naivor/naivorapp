naivorapp
===========
自己打算维护的一个快速android app开发框架

尚不完善，有些功能正在努力添加中。。。。

特点
===========
### 1.使用比较前沿的一些开发技术和库

    dagger2   rxJava   retrofit   butterknife  autovalue  okhttp  fresco
    
### 2.结合clean架构思想和MVP开发模式，进行app结构分层和代码解耦，方便扩展和维护
    
### 3.采用流行的Activity+Fragment+ViewPager搭建应用的主框架，常用工具类集成
    
### 4.会一直维护和完善，因为自己项目会用
